### Django 설치

- pip install django



### Django 프로젝트 생성

1. 프로젝트 생성
   - django-admin startproject [프로젝트명]
2. 서버 실행
   - 프로젝트 디렉터리 접속
   - python manage.py runserver
3. 서버 접속
   - 주소창에 127.0.0.1:8000 또는 localhost:8080 입력



### Django App 생성

- 프로젝트 디렉터리 접속
- python manage.py startapp [앱이름]
- Django에서 App이란 웹사이트를 기능별로 분류해놓은 단위를 뜻함
