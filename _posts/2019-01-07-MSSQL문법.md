# MS-SQL 문법

- 데이터베이스 조회
  - SELECT * FROM master.dbo.sysdatabases;
- 데이터베이스 생성
  - CREATE DATABASE [NAME];
- 데이터베이스 사용
  - USE [DATABASE_NAME];
- 전체 테이블 조회
  - SELECT * FROM information_schema.tables;
- 상위 N개의 데이터 조회
  - SELECT TOP N FROM [TABLE_NAME];
- 중복되지 않는 데이터 조회(오름차순 정렬)
  - SELECT DISTINCT [COLUMN] FROM [TABLE_NAME];
